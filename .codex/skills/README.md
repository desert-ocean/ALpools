# Project Skills

Каталог содержит project-specific skills для `ALpools`.

## Обязательные skills

- `coordinator`
- `analyst`
- `teamlead-gate`
- `architect`
- `domain-modeler`
- `backend-bot-engineer`
- `qa`
- `bug-triage`
- `stability-engineer`
- `customer-reporter`
- `context-pack-builder`
- `autopilot-maintainer`

## Принцип

Skills используются как role/runbook contracts. Если тип работы совпадает с skill, агент обязан опираться на него, а не импровизировать весь процесс заново.
