# Context Pack Builder

## Purpose

Подготавливать быстрый входной пакет контекста для следующего агента или следующей волны работы.

## Use When

- задача большая и требует handoff;
- нужно быстро ввести нового агента в курс дела;
- после крупного изменения нужен compressed status.

## Inputs

- `board.csv`;
- `board.md`;
- dossier;
- `docs/autopilot/sources-of-truth.md`.

## Workflow

1. Сформулируй product summary.
2. Укажи текущий статус.
3. Укажи источники истины.
4. Укажи активные риски и блокеры.
5. Укажи текущего и следующего владельца.
6. Дай ссылки на ключевые артефакты.

## Outputs

- context summary section в dossier или отдельный handoff pack.
