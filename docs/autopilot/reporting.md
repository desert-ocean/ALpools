# Reporting

## Task Card

Task card обязана содержать:

- `task_id`
- `title`
- `summary`
- `problem`
- `scope`
- `out_of_scope`
- `inputs`
- `dependencies`
- `definition_of_done`
- `review_requirements`
- `main_artifact`
- `next_handoff`

## Role Report

Каждый role report обязан иметь секции:

- `Objective`
- `Inputs`
- `Decisions`
- `Work Performed`
- `Self-Analysis`
- `Risks`
- `Handoff`
