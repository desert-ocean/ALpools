# Task Flow

## Базовый цикл feature/task

1. `todo`
2. `analysis`
3. `ready_for_lead`
4. `ready_for_arch` при необходимости
5. `ready_for_impl`
6. `in_progress`
7. `ready_for_qa`
8. `qa_review`
9. `ready_for_lead_gate`
10. `done`

## Обязательные артефакты

- `analysis` -> task card, dossier, `analyst.md`
- `ready_for_arch` -> `architect.md`
- `in_progress` -> `implementation.md`
- `qa_review` -> `qa.md`
- каждый переход -> `handoff.md`
